# Instructions for installing GovBridge into Tampermonkey
## Step 1: Install Tampermonkey
**Tampermonkey** is a browser extension that lets you run userscripts, which are small programs that can customize websites by adding features or modifying existing ones.

Install the extension for your browser here: https://www.tampermonkey.net

## Step 2: Find The Tampermonkey Logo On Your Browser Extension Bar
On the extensions bar of your browser, be sure that the Tampermonkey logo is pinned there. This is how it looks like: <img width="69" height="32" alt="image" src="https://github.com/user-attachments/assets/9427e028-e574-4298-bbfe-8572f3bcddb2" />

## Step 3: Open Dashboard
Click the Tampermonkey logo and choose "Dashboard".

<img width="244" height="289" alt="image" src="https://github.com/user-attachments/assets/337cc1a0-9d83-439a-8e4f-fa0f8fa3e70d" />

## Step 4: Open Utilities
Find the tabs to the right inside the Dashboard, click the tab labeled **Utilities**.

<img width="1000" height="110" alt="image" src="https://github.com/user-attachments/assets/3d718a20-e44b-47cb-a9dd-40a4a743fa1c" />

## Step 5: Add Script URL
At the bottom of **Utilities** tab, find ***Import from URL*** field. Copy and paste the following URL in that field: 
https://raw.githubusercontent.com/zen-citizen/GovBridge/refs/heads/main/GovBridge.js

After adding the URL, click *Install*.

<img width="1662" height="569" alt="image" src="https://github.com/user-attachments/assets/0dabc04d-9e16-4381-a095-d3a82d5d684b" />

## Step 6: Install the Script
The page reloads displaying the script details. Click *Install*

<img width="1713" height="902" alt="image" src="https://github.com/user-attachments/assets/0762be18-9b72-4020-b8c9-e1a761d93d21" />

## Step 7: Confirm Installation
Find and click the **Installed Userscripts** tab at the top of Tampermonkey

You should see *GovBridge – Seva Sindhu Login Helper* in the list. The toggle under the ***Enabled*** column should be Green if it is enabled.

<img width="1665" height="127" alt="image" src="https://github.com/user-attachments/assets/64fd50d0-9121-4e88-bf33-bcad37e01934" />

## Step 8: See the change
Go to [Seva Sindhu](https://sevasindhuservices.karnataka.gov.in), the Tampermonkey logo should show a red square with '1', like so: <img width="33" height="36" alt="image" src="https://github.com/user-attachments/assets/224e937b-0be9-4192-9c89-2db48e54d6aa" />

The page should load the improved UX.
